Everything You Need

Built for power users

Every feature is toggleable in Settings. Keep it minimal or turn everything on.

★

Rich Annotations

Star, highlight (5 colors), tag with custom labels, add Google Docs-style side comments. Browse all annotations across sessions in the sidebar.

⚡

Live Auto-Refresh

Watch conversations update in real-time as Claude responds. WebSocket-powered file watching with smooth scroll-to-bottom button.

/

Global Search

Search across all sessions in all projects. Filter by role, tool type, and date range. Keyboard-driven with arrow key navigation.

Δ

Diff View

Edit tool calls show red/green unified diffs instead of raw text. 3+ consecutive tool calls are grouped into collapsible summaries.

📊

Analytics

Daily cost charts, token usage, activity heatmaps, tool breakdown, model distribution. Filter by date range and per-project.

📁

Multi-Provider

Claude Code and OpenAI Codex in one interface. Collapsible provider sections. Pin or hide projects. Filter by provider.

Get Started

Up and running in 30 seconds

1

Install globally

npm install -g claude-journal

2

Start the daemon

claude-journal --daemon --port 5249

Runs in the background. Survives closing your terminal.

3

Open your browser

Go to http://localhost:5249 — your conversations are already there.

4

Start talking

Click any session and type in the floating input box. Claude resumes with full context.
