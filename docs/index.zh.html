功能一览

为高级用户打造

所有功能均可在设置中开关。保持简洁，或全部开启。

★

丰富的批注

收藏、高亮（5 种颜色）、自定义标签、侧边评论。在侧边栏中浏览所有会话的批注。

⚡

实时自动刷新

当 Claude 回复时，对话实时更新。基于 WebSocket 文件监听，带有滚动到底部按钮。

/

全局搜索

跨项目搜索所有会话。按角色、工具类型和日期范围过滤。支持键盘驱动导航。

数据分析

每日费用图表、token 用量、活动热力图、工具分布、模型占比。按日期范围和项目过滤。

📁

多平台支持

Claude Code 和 OpenAI Codex 统一界面。可折叠的平台分组。支持置顶或隐藏项目。

快速开始

30 秒即可启动运行

1

全局安装

npm install -g claude-journal

2

启动守护进程

claude-journal --daemon --port 5249

在后台运行，关闭终端也不会中断。

3

打开浏览器

访问 http://localhost:5249 — 你的对话记录已在那里等候。

4

开始对话

点击任意会话，在浮动输入框中输入。Claude 会带着完整上下文继续对话。
